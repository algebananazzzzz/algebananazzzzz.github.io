HTML basics

Most people consider HTML and CSS as NOT a programming language, those language has no logic, therefore its not programming. Simple. HTML is stupid, period. But one thing we know for sure, is that they are definitely a form of language, that the whole web architecture is based, be it chrome edge tor, all browsers must be able to interpret them to be even called browsers.

In this section, you will learn the basics of how HTML is structured.

What is HTML?

There are many descriptions of HTML, like how it stands for Hyper Text Markup Language, is the standard markup language for creating web pages and all that nonsense, I'll explain html by example.

This is a basic HTML template. In the world of computer science, we usually start any programming language with "hello world". It's like the first step you take as a baby.

<!DOCTYPE html>
<html lang="en" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title></title>
  </head>
  <body>
    <p>Hello World!</p>
  </body>
</html>

Now I want you to do this,
1. Open the notepad application in windows. Use the search feature.
2. Copy and paste the above code segment into it.
3. Save (control s) the file. We usually save our first file as index.html but anything with this syntax works: name.html. Save it in an easily accessible place like desktop.
4. Go to a browser and open (control o) the exact same file. And drumrolls, you'll see a "Hello world" on your screen!
5. Congratulate yourself, you are now a certified coder!!

So this is basically how the browser works. It reads your html file, be it from your computer or from a server, which I will teach you how to deploy later, and translates it into the web interface. Now, what if we want something else, lets say a image, or header etc? I want you to observe what we're doing here.

This is the exact code I use to express this sentence.

<p>This is the exact code I use to express this sentence.</p>

So basically, the <p> opening tag tells your browser that I want a paragraph component here (denoted by p). Of course, when you start a sentence, you gotta finish it. The browser can't pick up periods, so you gotta close the sentence using the closing tag, </p>

Try changing the p tag in your hello world file into a h1 tag! See what happens now.
