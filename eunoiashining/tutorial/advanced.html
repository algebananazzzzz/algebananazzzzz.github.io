Using the resume template

"Don't reinvent the wheel"

Building a webpage from scratch usually involves coding thousands of lines, and most likely, without proper design thinking etc., you'll end up creating a square wheel that makes the user go :< There are existing css frameworks, most prominently being Bootstrap that allows you to quickly build user friendly webpages, as people has thought together and put together resources to create these frameworks. And yes, like how we dress differently for different occasions, it's important for webpages to have proper CSS for different purposes. Like how we dress differently for different occasions, different webpages require different CSS for different purposes, which bootstrap takes care of it by providing free themes for it.

Okay enough theory. Go to this website and download the resume theme.

For this tutorial's purpose, due to time constrains, I will provide in depth tutorial on this design:

After you have downloaded, unzip and open the index.html file using notepad, and open the same file in the browser. You can edit your webpage by this way, and view changes at the same time.
