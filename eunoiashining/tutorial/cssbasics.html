CSS basics

Okay now, lets dive into styling yea?? I'll use the same approach I did wif html to teach ya'll.

This is a generic CSS code

.myclass {
   font-size: 1rem;
   color: #f50057
}

What happens is that we specified a class (myclass), and anything that has "myclass" specified in their class would take up the same style!

Styled paragraph

<p class="myclass">Styled paragraph</p>

This is one way we can specify style. But, for our purpose, we are using a preset template, we won't be creating our own css probably because its just too hard (trust me I tried, but it only suceeds sometimes). We will be using this syntax mainly.

Styled paragraph

<p style="font-size: 1rem; color:#f50057">Styled paragraph</p>

Font family

This may be confusing to many people... To do this, we gotta edit the css stylesheet, as well as the head of the html.

1. Go to this website and choose a font style.

2. Add the font, then copy the styling link.

3. Paste the styling link into the head of your html. Then, open the css file, search for the class that looks something like
body {
...
},
Add your specific style name into the font-family attribute, referring to the google site. So in my case,
body {
font-family: 'Architects Daughter', .....;
...
},
